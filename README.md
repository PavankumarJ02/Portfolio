# Pavan Kumar Jayavelu Portfolio

Static portfolio site built from resume content. It is ready to publish on GitHub Pages.

## Files

- `index.html` - main page
- `styles.css` - styling
- `script.js` - reveal animations and small UI behavior
- `assets/Pavan_Kumar_Jayavelu_Resume.pdf` - downloadable resume
- `assets/favicon.svg` - site icon
- `assets/og-card.svg` - social preview image
- `CNAME.example` - custom domain template for GitHub Pages

## Publish on GitHub Pages

1. Create a GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and the `/ (root)` folder.
6. Save and wait for GitHub Pages to publish the site.

## Use your `.me` domain

1. Rename `CNAME.example` to `CNAME`.
2. Replace the sample domain with your real `.me` domain.
3. In your domain provider DNS settings, point the domain to GitHub Pages.
4. In the repository `Pages` settings, confirm that the custom domain matches the file.

## Customize later

- Update text in `index.html`
- Change colors and spacing in `styles.css`
- Replace the resume PDF in `assets/`
